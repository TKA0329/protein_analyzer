How it works:
Put all the protein sequences in a csv file, like so (take care to stick to this format, with "sequence" at the top, and the sequences afterwards):
`
sequence
MPYEKHVEQTVVEKTEQGGSGGSYRHQTEAEAEKIRRELEKQGGGGSGGGGS
RQGGGAPAGGNIGGGQPQGGWGQPQQPQGGNQFSGGAQSRPQ

` 

Then upload it to the web app, and it should return a table with the properties for all sequences. This is probably what Yi Feng was asking about in the meeting. This helps us process a large number of sequences at once.
